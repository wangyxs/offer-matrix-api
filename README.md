# offer-matrix-api

[中文](./README.zh-CN.md) | [English](./README.en.md)

FastAPI backend for an AI mock interview and interview training platform.

面向 AI 模拟面试与岗位训练场景的 FastAPI 后端，提供认证、岗位管理、题目生成、语音能力、面试记录与分析等核心能力。

## Highlights

- Role-based interview training backend
- AI interview chat, question generation, and analysis
- Resume and material workflows for each target role

## 亮点

- 面向岗位化练习的面试训练后端
- 支持 AI 对话、题目生成与面试分析
- 支持简历与资料驱动的训练流程
